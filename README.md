# SMTP tester

## Simple server & tester for SMTP debugging

### Install basic

- clone the repos
- yarn
- in different terminals;
  - node index.js
  - node tester.js -h 127.0.0.1 -p 25

### Install with demo pipe

- clone the repos
- yarn
- in different terminals;
  - node index.js -p ./echojson
  - node tester.js -h 127.0.0.1 -p 25
